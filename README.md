## Project Overview

This project was created as part of the Digital Literacy course. The aim of this project is to understand how to use digital tools effectively, build a professional online presence, and stay safe while using the internet.

As a student digital ambassador, I explored different platforms and activities that help students become more aware and responsible in the digital world.

## How to Set Up and View This Project

> This is a documentation-based Digital Literacy Portfolio. There is no code to install or execute. Follow the steps below to access all project components.

**Prerequisites:** A web browser and a GitHub account (free). No software installation required.

**Step 1 — Clone or download the repository**
```bash
git clone https://github.com/palak25bce10920-ship-it/digital-literacy-project.git
```
Or click the green **Code** button on GitHub → **Download ZIP** and extract it.

**Step 2 — Navigate the folder structure**
```bash
cd digital-literacy-project
```
Open any `.md` file in a text editor, or view it rendered directly on GitHub by visiting the repository URL.

**Step 3 — View the project report**
Open `report/Project_Report.pdf` (or `.docx`) in any PDF viewer or Word-compatible application.

**Step 4 — Review task deliverables**
Each task folder (`task-1-presentation/` through `task-5-cybercrime/`) contains the relevant write-ups, screenshots, and checklists. Open them in order as described in the Module Summaries below.

**No dependencies. No environment setup. No execution required.** All deliverables are static files viewable in any browser or text editor.

## Module Summaries

**Task 1 – Digital Literacy Awareness Infographic (Module 1)**

Created a one-page visual infographic using a design tool to help batchmates understand what Digital Literacy is and why it matters. The infographic covers key topics including digital tools for students, safe internet practices, and professional online presence.

**Task 2 – Student Digital Portfolio (Module 2)**

Set up professional digital profiles on **GitHub**, **LinkedIn**, and **Kaggle**. Each platform serves a unique purpose — GitHub for code and projects, LinkedIn for professional networking and internships, and Kaggle for data science practice and competitions.

**Task 3 – Coding & Collaboration Platforms (Module 3)**

Completed a beginner-level coding challenge on **HackerRank** ("Say Hello, World! With Python") and created a **Digital Literacy Awareness Quiz** using Google Forms with 5 questions including multiple choice and short answer types.

Digital Literacy Awareness Quiz (Google Form):
[Click here to take the quiz](https://docs.google.com/forms/d/e/1FAIpQLSfO2MeIzgHyxukwGXwEsGKyX8MGidpJd86ABMhLU3swwYfQLg/viewform?usp=publish-editor)

**Task 4 – Professional Email & Etiquette Guide (Module 4)**

Drafted two professional emails — one requesting an assignment extension from a professor, and one expressing interest in a summer internship. Also created a Social Media Do's and Don'ts checklist for responsible online behaviour as a college student.

**Task 5 – Cybercrime Awareness Case Study (Module 5)**

Researched a cybercrime type (UPI / Online Payment Fraud) and wrote a detailed case study covering how it happens, who is targeted, and the consequences. Created a 'Stay Safe Online' prevention checklist with actionable tips specifically for college students in India, including UPI safety and reporting via cybercrime.gov.in and helpline 1930.
